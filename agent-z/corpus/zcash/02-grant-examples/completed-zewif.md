# Zcash Extensible Wallet Interchange Format (ZeWiF)

## Summary

A wallet interchange format enabling Zcash wallets to export and import wallet data in a standardised way. Collaboration between Blockchain Commons and Zingo Labs.

## Zcash alignment

Directly addresses wallet interoperability — a ZCG wishlist item. Reduces lock-in to individual wallet implementations. Benefits all Zcash wallet users.

## Problem

Zcash wallet users cannot easily move between wallet implementations. No standard format exists for exporting/importing wallet data including shielded keys and transaction history.

## Proposed solution

Create an extensible wallet interchange format specification and reference implementation that allows wallets to export and import data portably.

## Deliverables

Five milestones, all completed:
1. Format specification
2. Reference implementation
3. Integration with Zingo wallet
4. Documentation and developer guides
5. Final review and publication

## Milestones and budget

Five milestones, all completed and paid. KYC verified (grant exceeded $50k threshold).

## Forum/community response

Community supportive of wallet interoperability. No significant objections.

## Committee decision / status

Approved. All five milestones completed. Grant marked complete.
