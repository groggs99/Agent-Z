# ZCG Wishlist — What We're Ready To Fund

Source URL: https://zcashcommunitygrants.org/
Collected date: 2026-05-27
Source type: official ZCG homepage
Status: primary source, verbatim extraction.

## Framing

This wishlist is meant to encourage ideas for potential applicants to improve the Zcash ecosystem and is not exhaustive. If your idea to improve Zcash doesn't fit exactly, feel free to surprise us with your proposal.

## Open RFPs

- Zcash Lightwalletd Infrastructure Development and Maintenance (https://forum.zcashcommunity.com/t/rfp-zcash-lightwalletd-infrastructure-development-and-maintenance/47080)

## Wallets

- Zcash hardware wallet support
- Key-management tools
- Physical storage (paper wallets or lightweight hardware)
- Shielded wallets for smartphones (iOS/Android/GTK/Plasma Mobile)
- Web or Desktop wallets
- Payment URIs
- Easy one-click shielded payments
- Easy multi-sig tools for Shielded Addresses (e.g. an implementation of FROST in user-facing wallets)
- Point-of-sale for brick and mortar shops

## SDK

- Network layer privacy solutions for light client SDKs and libraries

## Core

- Developer documentation
- Unified Address support and implementation by exchanges
- R&D Decentralized light client syncing
- Audits for updated Zcash software

## Interoperability

- Atomic swaps
- Stablecoins on Zcash (ZUSD)

## Zapps

- Applications on top of the Zcash network
- Integrating native ZEC in end-user apps

## Ongoing Services

- Proving Key Distribution
- Light client servers / infrastructure

## Education

- Improving "Zcash Protocol Spec" for developers
- Websites with educational resources

## Agent Z usage rules

- When scoring grant fit, check the idea against each wishlist category above.
- An idea that maps directly to a wishlist item scores higher on alignment.
- An idea not on the wishlist is still eligible — ZCG explicitly says the list is not exhaustive.
- Open RFPs are the strongest signal of current committee interest.
- If the idea matches an open RFP, flag this prominently.
